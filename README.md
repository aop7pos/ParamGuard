# Agent 工具调用安全项目

## 项目目标

保护工具调用型 Agent，避免以下安全风险：

- 越权调用工具
- 工具参数被篡改
- 敏感信息泄露

## 第一版工具

第一版计划实现以下三个工具：

- 搜索文件
- 读取文件
- 模拟发送邮件

## 后续核心模块

后续将重点实现以下模块：

1. **意图绑定**  
   将用户的原始意图与 Agent 的工具调用行为进行绑定，防止 Agent 执行超出用户意图的操作。

2. **参数级授权**  
   对工具调用中的具体参数进行权限检查，例如限制可读取的文件路径、邮件接收人和发送内容。

3. **执行状态验证**  
   在工具执行前后检查执行状态，确保实际执行结果与预期行为一致。

4. **自动评测**  
   自动生成测试场景，评估 Agent 在越权、参数篡改和敏感信息泄露等攻击下的安全性。
